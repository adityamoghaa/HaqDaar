01 — Brief About the Idea

The Idea in One Sentence

HaqDaar bridges the gap between government welfare and the citizens who need it — using AI, voice, and multilingual NLP to make every scheme discoverable, understandable, and accessible.

🎯 The Problem We Solve

India's government has built an extensive social safety net — but it exists in bureaucratic PDFs, scattered across 50+ portals, written in English, inaccessible to the very people it was designed to serve. A first-gen student doesn't know which scholarship she qualifies for. A daily wage worker doesn't know he's eligible for free health insurance. The gap isn't eligibility — it's awareness and access.

💡 How HaqDaar Works

A citizen visits HaqDaar and simply talks — in Hindi, Tamil, Bengali, or English. Our AI extracts their profile silently from conversation, matches it against a curated database of 1,000+ schemes via a hybrid eligibility engine, and returns a personalised shortlist with plain-language explanations and step-by-step application guidance — voiced back through IndicTTS.

03 — Features

What HaqDaar Offers

🎙️

Multilingual Voice Interface

Whisper STT for voice input + IndicTTS for audio output. Speak in your regional language, hear the answer back. Built for low-literacy users.

🤖

Conversational Profile Extraction

LLM silently extracts state, income, age, caste, occupation from natural speech. No forms. Max 3 follow-up questions to fill gaps.

🎯

AI-Powered Scheme Matching

Hybrid rule-based + ML engine. Hard filters first (state, income, caste), then semantic re-ranking. Returns top 10–15 most relevant schemes.

📋

Plain Language Explanations

Every scheme explained simply: what you get, why you qualify, what documents you need, where to apply — no bureaucratic jargon.

🗺️

Step-by-Step Application Guide

Not just "you qualify." But "here's exactly what to do next" — CSC location, online portal link, form name, deadline.

📡

RAG-Backed Scheme Database

1,000+ schemes indexed as searchable embeddings. Weekly automated updates from myscheme.gov.in, state portals, and ministry websites.

📱

Progressive Web App (PWA)

Installable on any Android phone. Works on 2G. Offline-first for scheme browsing. No app store needed — just a browser link.

🔒

Privacy-First Architecture

Profile building and matching runs on-device. LLM only receives anonymized query. No login required. No personal data stored without consent.

🔖

My Schemes Bookmarking

Save matched schemes locally. Share via WhatsApp. Export as printable checklist for CSC visits. Revisit without re-entering profile.

🏛️

State + Central Coverage

Covers all 10 major scheme categories: scholarships, housing, health, farming, women, skills, startups, disabilities, senior citizens, food security.

08 — AMD Products & Resources

Usage of AMD in HaqDaar

AMD hardware is central to making HaqDaar's AI inference fast, cost-efficient, and scalable to millions of users across India.

⚡ AMD EPYC™ Processors

LLM Inference & API Serving

AMD EPYC 9004 series CPUs power the FastAPI backend and Claude API proxy layer. High core count + large memory bandwidth handles concurrent NLP inference requests from thousands of simultaneous users efficiently. EPYC's memory capacity (up to 6TB) allows caching the entire scheme database in RAM for sub-millisecond lookups.

🚀 AMD Instinct™ MI300X

Vector Embeddings & RAG Pipeline

The MI300X GPU accelerator handles the compute-intensive multilingual-e5 embedding generation for 1,000+ scheme documents. Its 192GB HBM3 memory makes it ideal for hosting the entire vector database in GPU memory, enabling near-instant semantic similarity search — critical for real-time scheme matching with sub-second response times.

🎙️ AMD ROCm™ Platform

Whisper STT Fine-tuning

ROCm open software platform enables fine-tuning of Whisper on Indian accent + regional language data using AMD GPUs. This improves STT accuracy for Bhojpuri, Maithili, and rural Hindi speakers — the very users who need HaqDaar most. ROCm's PyTorch compatibility makes migration from CUDA straightforward.

📱 AMD Ryzen™ AI

On-device Inference (Future)

AMD Ryzen AI NPUs in next-gen Android and Windows devices will enable HaqDaar's eligibility engine and compressed TTS models to run fully on-device — improving privacy, reducing latency, and making the app functional even with zero internet connectivity. Target for Phase 3 roadmap.
